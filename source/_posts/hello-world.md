欢迎来到安知鱼主题博客！这是一个基于 Hexo 框架 + anzhiyu 主题搭建的示例站点。

## 快速开始

### 创建新文章

```bash
hexo new "我的新文章"
```

### 本地预览

```bash
hexo server
```

### 生成静态文件

```bash
hexo generate
```

## 主题特性

- 暗色模式 / 阅读模式
- 自定义右键菜单
- AI 摘要
- 瀑布流相册
- 评论弹幕
- 更多特性请查看主题文档
