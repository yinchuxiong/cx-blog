# 🚀 启动指南

## 环境要求

- **Node.js** ≥ 10.13.0（推荐 v20+）
- **npm** 或 **yarn**

## 快速启动

### Windows 用户

双击 `start.bat` 即可自动安装依赖并启动。

或者在终端中：

```bash
cd E:\anzhiyu-blog
start.bat
```

### Mac / Linux 用户

```bash
cd /path/to/anzhiyu-blog
chmod +x start.sh && ./start.sh
```

---

## 手动启动

如果脚本无法运行，手动执行以下步骤：

```bash
# 1. 进入项目目录
cd anzhihu-blog

# 2. 安装依赖（首次或更新依赖后执行）
npm install

# 3. 启动开发服务器
npx hexo server -p 4000
```

启动后访问 **http://localhost:4000**

> ⚠️ 如果 SSL 证书报错（常见于公司网络环境），执行：
> ```bash
> export NODE_TLS_REJECT_UNAUTHORIZED=0   # Mac/Linux
> set NODE_TLS_REJECT_UNAUTHORIZED=0      # Windows CMD
> $env:NODE_TLS_REJECT_UNAUTHORIZED=0     # Windows PowerShell
> ```
> 然后再运行启动命令。

## 常用命令

| 命令 | 作用 |
|------|------|
| `npx hexo server -p 4000` | 启动本地预览 |
| `npx hexo new "文章标题"` | 创建新文章 |
| `npx hexo clean` | 清理缓存和生成文件 |
| `npx hexo generate` | 生成静态网站（部署前执行） |
| `npx hexo deploy` | 部署到服务器 |

## 目录说明

```
anzhiyu-blog/
├── _config.yml              ← Hexo 框架配置
├── _config.anzhiyu.yml      ← 主题配置（自定义所有功能）
├── source/
│   ├── _posts/              ← 文章 .md 文件放这里
│   ├── bangumis/            ← 追番页面
│   └── _data/               ← 追番数据缓存
├── themes/anzhiyu/          ← 安知鱼主题源码
├── scaffolds/               ← 文章模板
├── start.bat                ← Windows 一键启动脚本
├── start.sh                 ← Mac/Linux 一键启动脚本
└── public/                  ← 生成的静态文件（git 忽略）
```
